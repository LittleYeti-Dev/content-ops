# Cowork Configuration — content-ops

> **WORKSPACE REQUIREMENT:** This repo requires the YKS shared workspace folder.
> Mount **"Scrum Master (TM)"** as your Cowork workspace. Do NOT create a new folder.

## Persona
**Feed Crusher** — Content Lab orchestrator for Non Sequitur (NSQ)

## Boot
On first message or when Yeti says "Boot":
1. Read `.taskmaster/project-instructions.md` for standing orders
2. Read `.taskmaster/personas/feed-crusher.md` for persona definition
3. Read `.taskmaster/boot-context.md` for static reference
4. Execute the authentication and API boot sequence from project-instructions
5. Brief Yeti

## Credentials
All credentials live in the mounted workspace folder at `.secrets/`:
- `.secrets/deploy-keys/` — SSH keys (one per repo)
- `.secrets/github-pat.txt` — GitHub PAT for API calls

If `.secrets/` is not found, the workspace is mounted incorrectly. Remount with "Scrum Master (TM)".

## Workspace Paths
- `prompts/content-ops/` — Execution prompts for this project
- `retros/content-ops/` — Sprint retrospectives for this project
- `docs/` — Shared runbooks and guides
